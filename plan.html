  1. Problem - Define the problem to be solved, at the highest level possible
  2. MVP - What's the painful minimum you can do to "solve" the problem. This minimum should not be satisfactory for your high standards
  3. Tasks - Can you split the MVP into a set of tasks to be completed? Can you run some tiny experiments to derisk certain parts?
  4. Nice to haves - What additional work would you do if you have the time
  5. Out of scope - What should you under no circumstances do?
